Mortal Kombat (сложное)

Игра-файтинг для двух игроков. Два игровых персонажа, расположенных по краям экрана, проводят бой. 
Допустимы следующие действия персонажей: передвижение влево/вправо, прыжок, удар ногой, удар рукой. 
Возможны дополнительные действия при нажатии дополнительных комбинаций клавиш. 
При удачном ударе отнимаются очки здоровья другого персонажа; игра идёт до тех пор, 
пока у одного из персонажей не закончатся очки здоровья.

Предлагаемый вариант разбиения по классам:


***Класс "Боец" - содержит функции отрисовки и перемещения. Принимает ввод пользователя, 
управляющего данным персонажем, перемещает его и отображает в соответствии с его положением. 
При отрисовке можно использовать различные текстуры для разных положений. 
Для определения удачности удара может использоваться площадь пересечения определённых областей двух персонажей.


***Класс "Поле" - отвечает за отрисовку фона, на котором происходит бой.


***Класс "Игра" - логика игры, управление бойцами.


***Класс "Противник"  - недруг уимеющий драться.
